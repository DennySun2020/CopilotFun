> Reference: [github.com/features/copilot/plans](https://github.com/features/copilot/plans)

GitHub Copilot offers six subscription tiers — three for individuals and three for organizations — each with increasing capabilities, model access, and premium request quotas.

## Plan Details

### Free — $0/month
Best for getting started with AI-assisted coding.

- 2,000 code completions per month
- 50 agent mode or chat requests per month
- Access to Haiku 4.5, GPT-5 mini, and more
- Copilot in your terminal (CLI)
- Available on GitHub.com, GitHub Mobile, and supported IDEs

### Pro — $10/month
Everything in Free, plus:

- **Unlimited** inline code suggestions
- **Unlimited** agent mode and chats with GPT-5 mini
- 300 premium requests/month for the latest models
- Copilot code review (pull request reviews)
- Copilot cloud agent — assign work and get pull requests
- Access to models from Anthropic, Google, OpenAI, and more
- Custom instructions via `instructions.md`
- Option to purchase additional premium requests

### Pro+ — $39/month
Everything in Pro, plus:

- 1,500 premium requests/month (5x more than Pro)
- Access to **all** models, including Claude Opus 4.6
- Access to GitHub Spark

### Max — $99/month
Everything in Pro+, plus:

- 1,500 premium requests/month
- The most comprehensive individual plan with maximum capabilities

### Business — $19/user/month
Designed for teams and organizations:

- Everything from Pro for every seat
- 300 premium requests per user/month with option to buy more
- **IP indemnity** and enterprise data privacy protections
- User management and usage metrics dashboard
- Organization-wide policy controls
- Claude and Codex available on GitHub and VS Code
- No training on your code by default

### Enterprise — $39/user/month
Everything in Business, plus:

- 1,000 premium requests per user/month (~3.3x more than Business)
- Access to **all** models, including Claude Opus 4.6
- Access to GitHub Spark
- Natively integrated into GitHub.com
- Enterprise-grade knowledge base access via GitHub Mobile
- Option to purchase additional premium requests

## Premium Requests

Premium requests give access to the latest and most capable AI models. Usage varies by feature and model. All paid plans include an option to purchase additional premium requests beyond the included quota.

| Plan | Premium Requests |
|---|---|
| Free | 50/month |
| Pro | 300/month |
| Pro+ | 1,500/month |
| Max | 1,500/month |
| Business | 300/user/month |
| Enterprise | 1,000/user/month |

> **Note:** Option to purchase additional premium requests is not available to users who subscribe (or have subscribed) to Pro or Pro+ through GitHub Mobile on iOS or Android.

## Data & Privacy

- **Business and Enterprise** plans do **not** use your code to train AI models by default.
- **Free, Pro, and Pro+** users can opt out of AI model training via [GitHub settings](https://github.com/settings/copilot/features).
- Business/Enterprise prompts and suggestions in IDE (chat/completions) are **not retained**.
- All plans include filters for insecure code patterns (hardcoded credentials, SQL injection, path injection).
- Copilot includes an optional **code referencing filter** to detect suggestions matching public code on GitHub.
